## Fonctionnalites

- **62+ mods Fabric** pre-installes et preconfigures

- **8 resource packs** inclus (FurfSky Reborn, Hypixel+, Dark UI, etc.)
- **Shaders** Complementary Unbound inclus
- **Configs overridable** : base preconfiguree, modifiable librement par chaque joueur
- **Mise a jour automatique** du launcher

## Installation

1. **Telecharger** le [SkyBlock Launcher-setup-1.0.0.exe](https://github.com/matou4580-dev/SkyBlockLauncher/releases/latest)
2. **Installer** — lancer l'installeur, suivre les etapes
3. **Se connecter** avec son compte Microsoft
4. **Choisir son profil** (1080p / 1440p / 4K / Custom)
5. **Jouer** — cliquer sur Launch

> **Note :** Windows SmartScreen peut afficher un avertissement car l'application n'est pas signee. Cliquer sur "Informations complementaires" puis "Executer quand meme".

## Keybinds essentiels

| Touche | Action |
|--------|--------|
| `G` | GUI Editor (deplacer overlays) |
| `M` | Warp Menu |
| `K` | Toggle Shaders |
| `F1` | Wiki Fandom (survol item) |
| `F4` | Wiki Hypixel (survol item) |
| `F6` | Prix d'un item |
| `O` | Recherche d'items (SkyOcean) |
| `/sh` | Config SkyHanni |
| `/skyblocker config` | Config Skyblocker |

## Configuration requise

- **Java** : 17 ou superieur (21 recommande)
- **RAM** : 4 GB minimum alloues a Minecraft
- **OS** : Windows 10/11 (64-bit)
- **Compte** : Microsoft / Minecraft premium
